## PHP 라이프 사이클의 시작
- PHP는 PHP CLI 또는 PHP FPM 등의 모듈에 의해 실행된다.
- 이런 PHP 프로세스를 시작하기 위한 모듈을 시작하는 단계를 모듈 시작단계(the module startup step)라고 한다.
- 모듈 시작 단계를 MINIT 단계라고 표현함

## RINIT
- 리퀘스트가 들어 올 때마다 요청 시작 단계를 실행한다. 이를 RINIT이라고 부른다.
